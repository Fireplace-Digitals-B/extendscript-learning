JavaScript vs ExtendScript: A Time Travel Adventure! 🚀

Hey Dan! Ever felt like ExtendScript is JavaScript from another era? Well... you're not wrong! 😄

Let's explore why ExtendScript feels a bit... vintage, and how to make the most of it! 🎬

The Time Machine Effect 🕰️

Imagine you're a time traveler who knows modern JavaScript, but you've landed in the early 2000s. That's basically what using ExtendScript feels like! Here's why:

Modern JavaScript

// Modern JavaScript
const videoLayers = ['intro', 'main', 'outro'];
videoLayers.forEach(layer => {
    console.log(`Processing ${layer}...`);
});

// Using modern features
const settings = {
    duration: 5,
    effects: ['fade', 'blur']
};

const { duration, effects } = settings;

ExtendScript (Time Travel Mode)

// ExtendScript
var videoLayers = ['intro', 'main', 'outro'];
for (var i = 0; i < videoLayers.length; i++) {
    $.writeln("Processing " + videoLayers[i] + "...");
}

// No destructuring - old school way
var settings = {
    duration: 5,
    effects: ['fade', 'blur']
};

var duration = settings.duration;
var effects = settings.effects;

Key Differences to Remember 🗝️

📜

ECMAScript 3

ExtendScript is based on ES3 (circa 2000). No fancy arrow functions or template literals here!

🔄

var Only

Forget let and const - var is your new (old) best friend

📝

$.writeln()

console.log? Nope! We use $.writeln() for logging

🎯

After Effects Objects

Special objects like app and project are your gateway to AE

The Good News! 🌟

Even though ExtendScript feels like a blast from the past, it's still incredibly powerful for After Effects automation and effects. Here's what you can do:

  • 🎬 Control compositions programmatically
  • 🎨 Create and modify layers
  • ⚡ Apply and adjust effects
  • 🚀 Automate repetitive tasks

Your First ExtendScript Magic ✨

// Create a new composition
var comp = app.project.items.addComp(
    "My Awesome Comp",    // name
    1920, 1080,          // width, height
    1,                   // pixel aspect
    30,                  // duration (seconds)
    24                   // frame rate
);

// Add a solid layer
var solid = comp.layers.addSolid(
    [1, 0, 0],          // RGB color (red)
    "Red Solid",        // name
    comp.width,         // width
    comp.height,        // height
    1                   // pixel aspect
);

Quick Tips for Time Travelers 🧙‍♂️

  • 🔍 Always use var for variables
  • 📝 Use $.writeln() for debugging
  • 🎯 Keep your code simple and straightforward
  • 🚀 Test small chunks of code before building bigger scripts

Ready for Your First Mission? 🎯

Try running this simple script in After Effects to get started:

// Your first ExtendScript
alert("Hello After Effects!");
$.writeln("Script running...");

// Get project info
var project = app.project;
$.writeln("Project name: " + project.file ? project.file.name : "Unsaved Project");
$.writeln("Number of items: " + project.items.length);

💡 Pro Tip: Save this as a .jsx file and run it from After Effects' File > Scripts > Run Script File...
