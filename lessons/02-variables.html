Variables & Data Types in ExtendScript 📦

Let's learn how to store and work with different types of information in ExtendScript!

Think of variables as labeled boxes where we can store our stuff 📦

What are Variables? 🤔

Variables are like labeled containers that help us store information we want to use later. Just like how you might label a box "Christmas Decorations" to know what's inside, we label our data in ExtendScript!

Creating Variables 📝

// Creating a variable to store text
var projectName = "My Cool Animation";

// Creating a variable to store a number
var duration = 5;

// Creating a variable to store yes/no information
var isPlaying = true;

💡 In ExtendScript, we always use the word 'var' to create variables

Different Types of Information 🎨

📝

Text (Strings)

Words and sentences in quotes

var name = "Dan";
🔢

Numbers

Any number you can think of

var age = 25;
✅

Yes/No (Booleans)

true or false values

var isHappy = true;
📦

Lists (Arrays)

Collections of items

var colors = ["red", "blue"];

Using Variables in After Effects 🎬

Here's a simple example of how we can use variables to create a text animation:

Creating an Animated Text Layer ✨

// Store our text in a variable
var message = "Hello After Effects!";

// Store some settings
var textSize = 72;
var textColor = [1, 1, 1];  // White color

// Create the text layer
var comp = app.project.activeItem;
var textLayer = comp.layers.addText(message);

// Apply our settings
var textProp = textLayer.property("Source Text");
var textDocument = textProp.value;
textDocument.fontSize = textSize;
textDocument.fillColor = textColor;
textProp.setValue(textDocument);

📚 Glossary

variable
A container for storing information that we want to use later in our script.
string
Text that we put in quotes, like "Hello" or "My Animation".
number
Any number we want to use, like 42 or 3.14.
boolean
A yes/no value that can only be true or false.
array
A list of items we can store together, like ["apple", "banana", "orange"].
